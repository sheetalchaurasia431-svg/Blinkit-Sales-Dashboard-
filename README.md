# Blinkit Sales Dashboard | Power BI

##  Project Overview

This project is an interactive sales dashboard built using Microsoft Power BI to analyze Blinkit's sales performance. The dashboard provides insights into sales, customer ratings, outlet performance, and product categories through interactive visualizations and filters.

It enables users to monitor key business metrics and make data-driven decisions.



##  Dataset

The dataset contains information related to:

- Outlet Establishment Year
- Outlet Size
- Outlet Location Type
- Outlet Type
- Item Type
- Item Fat Content
- Sales
- Average Sales
- Customer Rating
- Number of Items



##  Tools & Technologies Used

- Microsoft Power BI
- Power Query
- DAX
- Microsoft Excel



##  Dashboard Features

- Interactive Filter Panel
- KPI Cards
  - Total Sales
  - Average Sales
  - Number of Items
  - Average Rating
- Outlet Establishment Trend Analysis
- Sales by Item Fat Content
- Sales by Item Type
- Sales by Outlet Size
- Outlet Type Performance Analysis
- Interactive Charts and Visualizations


##  Key Performance Indicators (KPIs)

- **Total Sales:** 508K
- **Average Sales:** 139.88
- **Number of Items:** 4K
- **Average Rating:** 3.93

---

##  Key Insights

- Analyzed overall sales performance across Blinkit outlets.
- Compared sales based on outlet size and outlet type.
- Identified the contribution of Low Fat and Regular products to total sales.
- Evaluated sales across different item categories.
- Tracked outlet establishment trends over time.
- Built an interactive dashboard with slicers for dynamic filtering.
